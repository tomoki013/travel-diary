# Changelog

このプロジェクトの主要な変更を記録します。
フォーマットは [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に準拠し、
バージョンは [Semantic Versioning](https://semver.org/lang/ja/) に従います。
採番ルールは [`docs/versioning-release.md`](docs/versioning-release.md) を参照してください。

> このバージョン履歴は、本リポジトリ（Next.js 版サイト）の実際のコミット履歴
> （2025-03 開始）から再構築したものです。利用者向けの公開更新履歴は
> `src/components/features/roadmap/UpdateList.tsx` と一致させています。

## [4.8.1] - 2026-06-22

### Changed

- ローディングアニメーションを刷新。初回ロード／リロード時は飛行機が航路を描き、サイト名とタグラインがせり上がるスプラッシュ（`InitialPreloader`、`window.load` 後にスライドアップで退場）。ページ遷移時は飛行機の航路だけ（`loading.tsx` の `PageLoader showText={false}`）。世界地図・検索などインラインの通常ローディングはやわらかい熱気球が漂う情景（`LoadingAnimation`）。いずれもサイトのアンバー基調・テーマ配色（`--foreground`/`--border`）と `font-heading` に合わせた（従来の紙飛行機 / mapRoute / luggageCarousel は破棄）。
- `/posts` 検索セクションの UI を統一・洗練。角丸はサイト基準の `rounded-md`。選択フィルタ（タグ・クイックアクセス・ページ送り）はアンバー、並び替えの選択は無彩色（セグメント風）、絞り込みボタンは控えめなアンバーのアウトラインに。ホバーは縁の色のみ変える控えめな挙動へ統一した（以前は黒/グレー/ティール/濃いアンバーが混在）。
- 並び替えを検索カードから独立させ、結果ヘッダー（ツールバー）に配置。モバイルはセレクト、PC はセグメントボタンと、画面幅で操作方法を出し分ける。
- 初回ロード／リロードのスプラッシュのブランド表記を、英字オーバーライン `TRAVEL DIARY` ＋サイト名「ともきちの旅行日記」＋タグラインの 3 段（順にせり上がる）に。
- 航路を進む要素を、簡素な三角から上面視の飛行機シルエット（柔らかな影付き）に変更。
- クイックアクセス（目的・エリア）を、選択中にもう一度押すと解除できるトグルに変更（人気のタグは従来どおりトグル）。
- 検索セクションの操作（並び替え・タグ・クイックアクセス・絞り込み・クリアなど）を `useOptimistic` / `useTransition` で楽観的に即時反映し、選択／解除の反応の遅さを解消。再取得中は結果一覧をスケルトン（カード型のプレースホルダー）で表示する。
- 検索バーの絞り込みボタンを、検索の真横から独立した行（説明文付き）へ移して役割を明確化し、配色をサイト基本のプライマリボタンに統一。
- 並び替えを結果ヘッダーの右端に固定し、「すべてクリア」は同じ行の左隣に表示（モバイル・PC とも同一行）。おすすめ以外を選ぶと並び替え（モバイルのセレクト）の位置がずれる問題を解消。
- 初回ロード／リロードのスプラッシュ背景のドットグリッドを撤去し、通常のサイト背景に統一。
- 絞り込みモーダルのフッターをモバイルで縦積みにしてレイアウト崩れを解消。条件が何も選択されていないときはリセットを表示しないようにした。
- 世界地図（ヒーロー・地域ページ）の読み込み中表示を、静的な大陸シルエット（`WorldMapPlaceholder`、削除）から通常のローディング（熱気球 `LoadingAnimation`）に変更。
- 記事の共有を OS の共有シートとリンクコピーのみに整理。

### Added

- フッターにアプリのバージョンを表示し、更新履歴（`/roadmap#updates`）へリンク。`package.json` の `version` を `next.config.ts` でビルド時に注入し、UI 側はそれを参照する（バージョン番号の手書き重複を排除）。
- 更新履歴の「NEW」表示を、現在のバージョン（`package.json`）と一致するエントリから自動判定するよう変更（`isNew` の手動付け替えを廃止し、並行作業での競合を低減）。

### Fixed

- 条件変更時に一度ページ先頭へ飛んでから結果へ戻る二重スクロールを解消（`router.push` の自動スクロールを無効化し、結果先頭へのスムーズ移動のみに）。
- タグ追加・並び替え・ページ送りなどの条件変更後のスクロール先を、検索セクション先頭から「結果の先頭（結果ヘッダー）」に変更。
- 絞り込みモーダル表示中に背景（body）がスクロールできてしまう問題を修正（表示中はスクロールを禁止）。
- `/posts` の検索バー横の絞り込みボタンの縦位置ずれを修正（検索バーと中央揃え）。
- `/posts` で表示されることのなかったデッドコードの `<Suspense fallback>`（`getAllPosts` 解決後に非サスペンドの `BlogClient` を包んでいた）を撤去。ページ遷移中のローディングは `loading.tsx` が担当する。

### Performance

- ローディングアニメーションを `left` / `width` などレイアウトを誘発するプロパティから `transform` / `opacity` 中心へ変更し、`prefers-reduced-motion` に対応。

### Removed

- 記事共有の SNS 個別ボタン（X / Facebook / LINE）を撤去（コンポーネントファイルも削除）。
- ローディング以外（装飾）だったパタパタ表示を `LoadingAnimation` から `SplitFlapBoard` として切り出し（見た目・挙動は従来どおり）。共通コンポーネントの責務をローディングのみに整理。

## [4.8.0] - 2026-06-22

### Added

- フッター（Resources）に公開更新履歴（`/roadmap#updates`）への「更新履歴」リンクを追加。`UpdateList` セクションに `id="updates"` アンカーを付与した。
- 検索・絞り込み・並び替えを一手に担う共通サービス `src/lib/searchService.ts`（`searchPosts`）を新設。`/api/search` と `/posts` が同一条件・同一適用順で結果を得るよう統一し、地域フィルタとキーワード文字数上限（`SEARCH_QUERY_MAX_LENGTH`）も共通化した。

### Changed

- `/posts` の検索セクションを再設計。絞り込みボタンを検索バーの真横（目立つ塗りスタイル）へ移動し、独立していた「並び替え」カードを検索セクション内へ統合した。
- `/posts` に「人気のタグから絞り込む」を追加し、選択状態を絞り込みモーダルのタグと連動させた（同じ `tags` state を共有）。
- 目的・エリアのクイックアクセス（プリセット）を「目的から探す／エリアから探す」の 2 列に整理し直して継続提供。人気のタグ（絞り込み連動）とは役割を分けた。
- 絞り込みモーダルの中身を整理。各項目に説明文と「N 件選択中」表示を追加し、項目間に区切り線を入れ、記事タイプ → カテゴリ → 実用ラベル → タグの順に並べ替えた。
- `/posts` の検索キーワードに文字数制限（`SEARCH_QUERY_MAX_LENGTH`、入力・サーバー両方）を追加。
- ヘッダー検索の候補取得を 5 件から 2 件へ削減（`API_MAX_RESULTS`）。「すべての結果を見る」ボタンは総件数で出し分けるよう修正した。
- About ページの構成を変更。「これまでの旅の軌跡」「Future Journey」を下方へ移し、「旅の記録（My Footprints）」を独立セクション（`FootprintsSection`）として最下部に配置した。

### Removed

- About ページの「一緒に旅をしませんか（Let's Journey Together）」セクション（`CommunitySection`）を撤去した。

## [4.7.1] - 2026-06-22

### Changed

- 記事ページの日付表示を「訪問日・公開日・更新日」で明確に区別。ヘッダーに公開日と更新日（設定がある場合のみ）を、記事末尾の自己紹介カードに訪問日と最終更新日を表示するよう整理した。
- 自己紹介カードの「最終更新」を、固定文言（記事公開時）から実際の更新日（`updatedAt`、未設定時は公開日）へ修正。
- `CLAUDE.md` のフロントマター例を実スキーマ（`publishedAt` / `updatedAt` / `travelDates` ほか）に更新。
- 右下のフローティング目次ボタンを、本文（記事 body）を読んでいる間だけ表示するよう変更（本文に入る前・読み終えた後は非表示）。出入りにスプリングアニメーションを付与。
- 記事カード（PostCard）の画像とラベル（ラベル・記事タイプ等）の間に余白を追加し、近すぎた間隔を調整。

### Removed

- ラベルと表示値が一致していなかった `getDatePrefix`（`src/lib/dateFormat.ts`）を撤去。
