# Performance Notes

このディレクトリは、ともきちの旅行日記のパフォーマンス改善に関する知見・計測結果・実験ログを蓄積する場所です。

## 方針

このブログは旅行体験を重視するため、以下は維持します。

- 高品質な写真
- アフィリエイト導線
- リッチなUI
- 記事末尾の回遊導線
- PWAや検索などの便利機能

一方で、以下を改善対象とします。

- 初期表示に不要なJavaScript
- 外部スクリプトの読み込みタイミング
- スクロール中の重い処理
- クリック後の反応遅延
- ページ遷移開始までの遅さ
- 不要な依存関係
- クライアントコンポーネントの過剰化

## 今回やらないこと

- 画像品質を極端に落とす
- アフィリエイト導線を削除する
- Framer Motionの全面撤去
- next/image の sizes 追加
