# Note 変換専門ルール (Note Conversion Rules)

Note の記事（下書き）を本ブログ形式に移行・変換するための特化ルールです。

## 1. Note 特有の要素変換

Note 特有の記法や要素を、以下のように変換・削除する。

- **スキ（Like）ボタン:** 「スキしてね」「応援お願いします」系のテキストは全て削除。
- **Note特有の埋め込み:** Noteカードや外部Note記事の埋め込みリンクは全て削除。
- **ハッシュタグ:** 文末のハッシュタグ（例: `#旅行 #一人旅`）は全て削除し、必要なものは Frontmatter の `tags` にのみ含める。
- **改行過多:** Note独特の「1文ごとの空行」が多すぎる場合、文脈に応じて適度に詰める。

## 2. Note ノイズの特定キーワード

変換の際、以下のキーワードが含まれる行は削除対象となる。

- 「すべて表示」
- 「スキありがとうございます」
- 「Noteで読む」
- 「フォローする」

## 3. 変換後の連携

変換が完了した後は、必ず `REVISION_RULES.md` に従って、共通のクリーンアップ（画像説明の除去など）を実行する。
