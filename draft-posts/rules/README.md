# Blog Rules & Guidelines

このディレクトリには、ブログ記事の執筆、編集、およびメタデータ管理に関する全てのルールが格納されています。AI（Gemini, Claude）はこのルールを「絶対的規範」として遵守します。

## ファイル構成

| ファイル名                     | 役割                                                                                                        |
| :----------------------------- | :---------------------------------------------------------------------------------------------------------- |
| **`REVISION_RULES.md`**        | **修正専門ルール。** 既存の下書きや外部テキストを、ブログ形式に適合させるための構造化とノイズ除去のルール。 |
| **`NOTE_CONVERSION_RULES.md`** | **Note変換専門ルール。** Noteの記事を本ブログに移行する際、Note特有の不要要素を除去するための特化ルール。   |
| **`METADATA_RULES.md`**        | **メタデータ運用ルール。** Frontmatterの型、必須項目、カテゴリ別の詳細な定義。                              |
| **`NAMING_CONVENTIONS.md`**    | **命名・表記ルール。** ファイル名の規則、数値・記号、地名の表記の統一ルール。                               |
| **`WRITING_GUIDELINES.md`**    | **執筆ガイドライン。** 執筆者のペルソナ、読者ターゲット、基本的な記事構成のガイド。                         |
| **`README.md`**                | (本ファイル) ルールディレクトリの概要と適用フロー。                                                         |

## ルールの適用フロー

1. **Research:** `METADATA_RULES.md` と `NAMING_CONVENTIONS.md` を確認。
2. **Writing:** 新規執筆時は `WRITING_GUIDELINES.md` に従う。
3. **Revision:** 下書き修正時は `REVISION_RULES.md` を適用。
4. **Note to Blog:** Noteからの移行時は `NOTE_CONVERSION_RULES.md` を経てから修正プロセスに入る。
