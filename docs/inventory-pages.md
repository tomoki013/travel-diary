## 要対応ページ詳細

### `/roadmap` — 廃止/noindex推奨

- **懸念**: 「Future Roadmap」「To Be Continued...」など未完成・未定感の強い表現が並ぶ。AdSense審査担当者に「サイトが未完成」と判断されるリスクがある。
- **推奨対応**: `robots: { index: false }` を追加してnoindex化する。または削除。
- **移行時**: Vercel移行後もnoindexのまま維持。

### `/affiliates` — 統合検討

- **懸念**: アフィリエイトポリシーの説明ページ単体で存在している。`/travel-essentials` もアフィリエイト系ページのため、2枚のアフィリエイト関連ページが存在する状態。
- **推奨対応**: `/editorial-policy` 内に統合するか、`/about` に「このサイトについて・アフィリエイトポリシー」としてまとめる。
- **移行時**: 統合前はそのまま維持。統合後は301リダイレクト設定要。

### `/about` — 内容確認推奨

- **現状**: 確認できていないが、AdSense審査でE-E-A-T（経験・専門性・権威性・信頼性）の根拠になる重要ページ。
- **推奨対応**: 運営者プロフィール・旅行経験の概要・サイトの目的が明確に書かれているか確認。不足があればリライト。
