# ページマップ

## 主なページの役割

- `/`: 記事へ入るための玄関
- `/posts`: 自分に合う入口を見つける一覧
- `/posts/[slug]`: 本文を読み、次に読む記事へ自然につなぐ
- `/destination`: 地域別に記事を探す
- `/gallery`: 写真から地域や記事へ入る
- `/journey/[id]`: 1 回の旅を時系列で追う
- `/series/[slug]`: まとまりで読む
- `/faq`: サイトの使い方と、よくある質問を整理する

## 情報設計の優先順位

1. ブログ記事
2. 記事の探し方
3. シリーズ・地域・旅のまとまり
4. Gallery / Globe などの補助機能

## 変更時に見るべき観点

- 記事導線が最優先のままか
- 重複した CTA が増えていないか
- 関連記事や次導線の文脈がずれていないか
- 補助機能が記事回遊を邪魔していないか
